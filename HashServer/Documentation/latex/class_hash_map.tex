\doxysubsection{Detailed Description}
Implements hash map. 

\mbox{\hyperlink{class_hash_map}{Hash\+Map}} class has hash function that maps key values to bins (positions in vector). Each element of the vector if a linked list, and elements are pushed back it the list.
