\doxysubsection{Detailed Description}
Server that accepts and hadles user requests. 

\mbox{\hyperlink{class_hash_server}{Hash\+Server}} accepts asynchronically a user connection, creates \mbox{\hyperlink{classcon__handler}{con\+\_\+handler}} object. After handling in in \mbox{\hyperlink{classcon__handler}{con\+\_\+handler}} class, it invokes handle\+\_\+accept. This function that checks for errors and reconnects if they are found, otherwise continues accepting.
