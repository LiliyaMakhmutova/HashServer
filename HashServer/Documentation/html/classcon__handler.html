Detailed Description

Handles each user connection, reads socket request and writes response to the socket.

HashServer accepts a user connection, creates con_handler object. This object asynchronically reads a socket, then if successful, parses request and asynchronically writes response to the socket. If unsuccessful, prints error to the stderr. Prints dubug messages is VERBOSE mode is on.
